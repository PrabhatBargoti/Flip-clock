# Flip Clock

A modern and minimal productivity clock built with React. It combines a flip clock, focus timer, ambient music, productivity statistics, and customizable themes into one distraction-free workspace.

---

## Features

- Flip clock with smooth paper-flip animation
- Focus timer (Pomodoro)
- Multiple focus modes
- Ambient music player
- Daily and weekly productivity statistics
- Focus streak tracking
- Goal progress tracking
- Custom themes
- Responsive design
- Local data storage
- Keyboard shortcuts *(planned)*

---

## Focus Modes

- Focus
- Short Break
- Long Break
- Study
- Coding
- Reading
- Workout
- Custom Mode

Each mode can have its own timer, colors, and music.

## How It Works

The application helps users stay focused by combining time management with a clean interface.

- Choose a focus mode.
- Start the timer.
- Play ambient music or your own playlist.
- Complete sessions.
- Track your daily and weekly progress.
- Customize the experience with themes and settings.

All user data is stored locally in the browser, making the application fast and usable without an account.
